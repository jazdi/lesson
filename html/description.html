Образец списка определений

TCP
Протокол управления передачей (Transmission Control Protocol) — это метод (протокол), используемый совмест-но с межсетевым протоколом (IP, Internet Protocol) для передачи данных в виде частей сообщений, называемых пакетами, между компьютерами через Интернет.
IP
Межсетевой протокол — это метод или протокол, с помощью которого данные пересылаются с одного компьютера на другой через Интернет. Каждый компьютер в сети Интернет имеет уникальный идентификатор в виде IP-адреса.
FTP
Протокол передачи файлов — это протокол, используемый для обмена файлами между компьютерами в Интернете.
HTTP
Протокол передачи гипертекстовых файлов — это протокол, используемый для обмена текстом, графическими изображениями, звуком, видео и другими мультимедийными файлами в сети.
